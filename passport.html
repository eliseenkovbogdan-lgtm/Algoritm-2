Извлечение данных из DOM

На этой странице мы извлекаем данные из DOM-структуры страницы с паспортными данными.

Извлеченные данные:

Фамилия: -
Имя: -
Год рождения: -

Как это работает?

Мы используем JavaScript для доступа к элементам DOM и извлечения данных:

  • Находим элементы с паспортными данными по их структуре
  • Извлекаем текстовое содержимое нужных элементов
  • Отображаем извлеченные данные на странице

Пример кода для извлечения данных:

// Извлекаем фамилию (первая строка паспорта)
const lastName = document.querySelector('.passport-row:nth-child(1) .passport-value').innerText;

// Извлекаем имя (вторая строка паспорта)  
const firstName = document.querySelector('.passport-row:nth-child(2) .passport-value').innerText;

// Извлекаем дату рождения (пятая строка паспорта)
const birthDate = document.querySelector('.passport-row:nth-child(5) .passport-value').innerText;

// Извлекаем год из даты рождения (формат DD.MM.YYYY)
const birthYear = birthDate.split('.')[2];

Скриншот извлечения данных через Инструменты разработчика

Для получения данных через Инструменты разработчика можно использовать:

document.getElementsByClassName('passport-value')[0].innerText // Фамилия
document.getElementsByClassName('passport-value')[1].innerText // Имя  
document.getElementsByClassName('passport-value')[4].innerText // Дата рождения
